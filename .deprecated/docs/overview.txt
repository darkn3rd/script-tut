This is a general tutorial of scripting modeled after typical tasks one would do with shell programming.  Thus it is oriented toward common tasks in system administration.

 - Joaquin Menchac (c) 2014

These are the categories.

A. Output
   - simple output to stdout
B. Variables
   - array declaration of number, characters, strings
C. Arithmetic
   - basic integer arithmetic
   - floating point arithmetic
   - math functions
D. Input
   - simple input from stdin
E. Branch
   - if on number
   - case on single character
   - if on single character
F. Looping
   1. iterative loop 
      - example: 10 to 1
   2. conditional loops
   3. collection loop
      - iterate through set of items 
      - example: directoy listing
G. Arrays
   1. Array Initialization
      - initialize array by index
      - array length
      - enumerate all elements
   2. Array Enumeration 
      - declare and initialize array
      - enumerate array by collection loop
   3. Array Enumeration
      - declare and initialize array
      - enumerate array by index
H. Associative Arrays
   1. Associative Array Initialization
      - initialize associative array by key
      - enumerate all keys
      - enumerate all values
   2. Assoication Array Enumeration
      - declate and initizliae associative array
      - merge two associative arrays
      - enumerate associative array by key
